**Python**

Python programming language is definetly one of the most popular programming languages due to the following reasons:

  • It is readable. In comparison to other languages such as C, Python has a simpler syntax, making it relatively easy to comprehend even for non-programmers. This enables you to concentrate on the problem rather than the code.
  • Python is interpreted. Basically this means that you do not have to run your code through a compiler like most high level programming languages to enable the CPU understand the instructions.
  • Python is FREE!!
  • Python allows you to write and run a small part of a program (in cells) without having to write the whole program . This saves a lot of time and reduces the chances of getting run-time errors whilst making it easier to debug.
